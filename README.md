# image_colors
Detects the n most common colors within an image.

--------
### Usage
```bash
./image_colors /path/to/picture <num_colors>
```
or
```bash
cargo run /path/to/picture <num_colors>
```

### Example
```bash
./image_colors /diamond.png 5
#282C34 - 1901447 pixels
#C678DD - 12002 pixels
#51AFEF - 11158 pixels
#A9A1E1 - 10638 pixels
#292D35 - 5673 pixels
```
